## :books: Description

Holidaze is a modern accommodation booking application developed as part of **Project Exam 2 at Noroff**.

The goal of this project was to apply the skills learned over two years of frontend development to build a complete booking platform using an existing API.

The application includes both:

- A **customer-facing experience** for browsing and booking venues
- An **admin/venue manager interface** for managing venues and bookings

### :sparkles: Features

### Public users

- Browse all venues
- Search venues by name or location
- View individual venue pages
- View availability calendar

### Registered users

- Register with `stud.noroff.no` email
- Login/logout
- Update avatar
- Create bookings
- View upcoming and past bookings
- Edit or delete bookings (if not past)

### Venue managers

- Create venues
- Edit venues
- Delete venues
- Add multiple images per venue
- Preview venues before publishing
- View bookings for owned venues

### Booking experience

- Select date ranges with calendar
- Prevent booking unavailable dates
- View total price before booking

## :construction_worker: Getting Started

Before you begin, make sure you have the following installed:

- [Node.js](https://nodejs.org/)
- A terminal
- Your preferred code editor

### Clone the Repository

```bash
git clone git@github.com:VildeAvloes/project-exam-2.git

```

### Navigate to project folder

```
cd project-exam-2
```

Install dependencies

```
npm install
```

## Usage

When installed you can use the following commands to run the application

**Start:** Launches the application in web browser and watch for changes.

```
npm run start
```

**Build:** Builds the application for production

```
npm run build
```

**Format:** Formats files using Prettier

```
npm run format
```
